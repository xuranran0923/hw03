# 人脸识别项目说明

## 1. 项目结构
- 人脸识别.py        主程序代码
- requirements.txt    依赖环境清单
- known_faces/       人脸库（存放已知人脸图片）
- 运行截图.png       效果展示

## 2. 功能说明
本项目实现：
1. 上传图片自动检测人脸位置
2. 与人脸库（known_faces）中的图片进行比对
3. 框选出人脸并显示识别结果
4. 基于Web网页展示

## 3. 检测识别流程
1. 加载人脸库中的人脸特征
2. 上传待识别图片
3. 检测图片中的所有人脸
4. 与人脸库进行匹配对比
5. 输出识别结果并标注姓名

## 4. 人脸库准备方法
1. 在项目目录下创建 known_faces 文件夹
2. 放入清晰正脸照片，文件名为对应姓名（如：zhangsan.jpg）
3. 程序会自动加载并建立人脸库

## 5. 运行环境与命令
### 安装依赖命令
pip install -r requirements.txt

### 项目运行命令
streamlit run 人脸识别.py

### 访问方式
本地访问地址：http://localhost:8501
端口号：8501
